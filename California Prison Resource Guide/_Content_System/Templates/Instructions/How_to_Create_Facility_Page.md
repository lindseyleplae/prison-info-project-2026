# How to Create a Facility Page

This guide walks you through creating a new facility page for any California state prison.

---

## Before You Start

1. Read the **STYLE_GUIDE.md** to understand tone and language rules
2. Open **TEMPLATE_Facility_Page.docx** as your starting point
3. Gather the information listed in the "Information to Collect" section below

---

## Information to Collect

Before writing, gather this information for the facility:

### Basic Information
- [ ] Official facility name
- [ ] Facility code (e.g., SQ for San Quentin)
- [ ] Physical address
- [ ] Mailing address (may be different)
- [ ] Main phone number
- [ ] Visiting office phone number

### Visiting Information
- [ ] Visiting days and hours
- [ ] How to schedule (VSA, phone, email?)
- [ ] Dress code specifics (if stricter than standard)
- [ ] Locker availability
- [ ] Photo availability and pricing

### Communication
- [ ] Current phone/video provider (ViaPath, Securus, etc.)
- [ ] Any facility-specific communication rules

### Programs (if notable)
- [ ] Education programs
- [ ] Vocational programs
- [ ] Special programs worth mentioning

### Practical Tips
- [ ] Nearby hotels/motels
- [ ] Gas stations
- [ ] Places to eat
- [ ] Any local quirks visitors should know

---

## Where to Find Information

### Official Sources (Primary)
1. **CDCR Website** — cdcr.ca.gov/facility-locator/
   - Basic facility info, addresses, phone numbers
2. **Facility's own page** on CDCR site
   - Sometimes has visiting schedules
3. **Title 15 Regulations** — For official rules

### Community Sources (Secondary)
Use these to supplement, but verify when possible:
1. **Prison Talk forums** — prisontalk.com
2. **Reddit** — r/ExCons, r/Prison, state-specific subs
3. **Facebook groups** — Family support groups for specific facilities

**Important:** When using community sources, frame tips as "visitors report that..." rather than stating as fact.

---

## Writing the Page

### Step 1: Header Section
Fill in:
- Facility name
- Location (city, region)
- Facility type (Level I-IV, etc.)
- Quick facts box with key info

### Step 2: Location & Directions
- Address
- Driving directions from major highways
- Parking information
- Nearby amenities (brief)

### Step 3: Visiting Information
- Link to California Visiting Guide for general rules
- Facility-specific schedule
- How to schedule visits
- Any facility-specific rules

### Step 4: Communication
- Link to California Phone Guide for general info
- Current provider
- Any facility-specific details

### Step 5: Mail
- Link to California Mail Guide
- Facility mailing address
- Any special rules

### Step 6: Sending Money
- Link to California Money Guide
- Any facility-specific options

### Step 7: Programs (Optional)
- Only include if there's something notable
- Education, vocational, special programs

### Step 8: Tips from Visitors
- Practical tips (parking, food, timing)
- Frame as "visitors report..." or "some visitors suggest..."

### Step 9: Contact Information
- All relevant phone numbers
- Organized by purpose (visiting, general, medical, etc.)

---

## Quality Check

Before finalizing, verify:

- [ ] All placeholders are filled in (no [FACILITY NAME] left)
- [ ] Phone numbers are formatted consistently
- [ ] Addresses are complete
- [ ] Links to California Guides are included
- [ ] Language follows Style Guide (no "loved one" overuse, no therapy speak)
- [ ] Sources are noted for any specific claims
- [ ] "Last updated" date is included

---

## File Naming

Save as: `CONTENT_[Facility_Name]_Facility_Page.docx`

Examples:
- CONTENT_San_Quentin_Facility_Page.docx
- CONTENT_CIM_Chino_Facility_Page.docx
- CONTENT_Pelican_Bay_Facility_Page.docx

---

## Notes on Complex Facilities

Some facilities have multiple units (like CIM Chino with facilities A, B, C, D). In these cases:
- Create one page for the overall facility
- Include a section explaining the different units
- List different mailing addresses if applicable
- Note which units have different rules
