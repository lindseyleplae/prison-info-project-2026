# Quality Checklist
## Pre-Publish Review for All Content

Use this checklist before publishing any new page or major update.

---

## Content Review

### Language & Tone
- [ ] No overuse of "loved one" — language is varied (they/them, the incarcerated person, etc.)
- [ ] No therapy speak (hard, overwhelming, give yourself grace, you're not alone)
- [ ] No emotional framing (frustrating, difficult, disappointing)
- [ ] No parenting advice (how to talk to kids, don't punish them, etc.)
- [ ] No legal advice (information only, not "you should" legal statements)
- [ ] Tone is factual and supportive, not preachy

### Accuracy
- [ ] Information is from reliable sources (see Source Guide)
- [ ] Dates and deadlines are current
- [ ] Phone numbers are correct and formatted consistently
- [ ] Addresses are complete
- [ ] Vendor information is current (check vendor websites)
- [ ] Links work

### Completeness
- [ ] All sections from template are filled in (or intentionally omitted)
- [ ] No placeholder text remaining ([FACILITY NAME], TBD, etc.)
- [ ] Contact information is included
- [ ] "Last updated" date is included

---

## Formatting Review

### Structure
- [ ] Headings are clear and descriptive
- [ ] Information flows logically
- [ ] Related information is grouped together
- [ ] Page isn't too long (break up if needed)

### Callout Boxes
- [ ] Correct box type for content (warning, tip, info, etc.)
- [ ] Box titles are factual, not emotional
- [ ] Boxes are used for important info, not overused

### Consistency
- [ ] Formatting matches other pages on site
- [ ] Font and styling are consistent
- [ ] Bullet point style is consistent
- [ ] Spacing looks right

---

## Technical Review

### For Website
- [ ] Page title is clear and SEO-friendly
- [ ] Links to other site pages work
- [ ] External links open in new tab
- [ ] Mobile-friendly (check if possible)

### For Documents
- [ ] File name follows convention (CONTENT_[Name].docx)
- [ ] Saved in correct folder
- [ ] PDF version created if needed (for printables)

---

## Final Steps

- [ ] Read through one more time as if you're a visitor seeing it for first time
- [ ] Check for typos and grammar
- [ ] Verify most time-sensitive information one more time
- [ ] Add "Last updated: [Month Year]" if not present
- [ ] Save final version

---

## Quick Reference: Common Issues

| Issue | How to Fix |
|-------|-----------|
| "Your loved one" appears 5+ times | Vary: they, the incarcerated person, them |
| "This is hard/frustrating" | Remove or reframe factually |
| "It's okay to feel..." | Remove entirely |
| "You're not alone" | "Thousands of families go through this" |
| Vague box title like "A Note" | Make specific: "Documentation Required" |
| Outdated vendor info | Check vendor website, update |
| Phone number with no label | Add context: "Visiting Office: 555-1234" |
