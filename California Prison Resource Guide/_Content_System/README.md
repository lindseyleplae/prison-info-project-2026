## Quick Start: Creating New Content

### For a New Facility Page
1. Read STYLE_GUIDE.md
2. Open Templates/TEMPLATE_Facility_Page.docx
3. Follow Templates/Instructions/How_to_Create_Facility_Page.md
4. Run through QUALITY_CHECKLIST.md before publishing

### For a New State (Future)
1. Read STYLE_GUIDE.md
2. Research that state's DOC system
3. Create parallel structure to California content
4. Adapt — don't just copy and change state names

## Key Rules (Summary)

### Language
- Vary how you refer to incarcerated people (not just "loved one")
- No therapy speak (hard, overwhelming, grace, etc.)
- No parenting advice
- No legal advice

### Tone
- Factual and direct
- Supportive but not preachy
- Practical, action-oriented

### Sources
- Official sources first (DOC websites, regulations)
- Vendor websites for their services
- Community sources for practical tips (frame carefully)
