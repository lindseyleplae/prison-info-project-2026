# Source Guide
## Where to Find Information & How to Verify It

This guide explains where to find information for content and how to handle different source types.

---

## Source Hierarchy

Use sources in this order of preference:

### Tier 1: Official Sources (Most Reliable)
These are authoritative and should be your primary sources.

| Source | What It's Good For | URL |
|--------|-------------------|-----|
| CDCR Website | Policies, facility info, forms | cdcr.ca.gov |
| Title 15 (CA Code of Regulations) | Official rules, visitor rights | govt.westlaw.com or CA.gov |
| CDCR Operations Manual | Internal procedures | (request via public records) |
| Facility pages on CDCR | Specific facility details | cdcr.ca.gov/facility-locator |

### Tier 2: Vendor Sources (Reliable for Their Services)
These are accurate for their own products/services.

| Source | What It's Good For | URL |
|--------|-------------------|-----|
| ViaPath (GTL) | Phone, video, messaging | connectnetwork.com |
| Securus | Phone, video (some facilities) | securustech.net |
| JPay | Tablets, email, money | jpay.com |
| Access Corrections | Money transfers | accesscorrections.com |

### Tier 3: Advocacy Organizations (Good Context)
Useful for understanding issues; verify specific claims.

| Source | What It's Good For |
|--------|-------------------|
| Prison Policy Initiative | Research, statistics, policy analysis |
| ACLU (state chapters) | Rights information, legal challenges |
| Ella Baker Center | California-specific advocacy |
| Worth Rises | Telecom/vendor pricing research |

### Tier 4: Community Sources (Use Carefully)
Helpful for practical tips; always frame appropriately.

| Source | What It's Good For | Caveats |
|--------|-------------------|---------|
| Prison Talk (prisontalk.com) | Real visitor experiences | Info may be outdated; verify |
| Reddit (r/Prison, r/ExCons) | Current experiences, tips | Anonymous; verify when possible |
| Facebook groups | Facility-specific tips | Anecdotal; experiences vary |

## How to Use Community Sources

Community sources (Reddit, forums, Facebook) provide valuable practical knowledge that official sources don't cover. Use them, but carefully.

### Do:
- Use for practical tips (parking, what to bring, timing)
- Frame as "visitors report that..." or "some families have found..."
- Cross-reference with multiple sources when possible
- Note when information might vary

### Don't:
- State forum tips as official policy
- Copy information about rules without verifying
- Use outdated posts (check dates)
- Include information that could get someone in trouble if wrong

### Example of Good Framing:
> "Visitors report that arriving by 7:00 AM on Saturdays helps avoid long lines, though wait times vary."

### Example of Bad Framing:
> ~~"You need to arrive by 7:00 AM or you won't get in."~~

## Verifying Information

### For Policy/Rules
1. Check CDCR website first
2. Look for the specific regulation in Title 15
3. If community source says something different than official source, go with official
4. Note if there's a gap between official policy and common practice

### For Practical Information
1. Check multiple community sources
2. Look for recent posts (within last 6-12 months)
3. If multiple people report the same thing, it's likely accurate
4. Note that things may vary or change

### For Vendor Information
1. Go directly to vendor website
2. Call vendor customer service if needed
3. Vendors change frequently — note the date you verified

---

## When Sources Conflict

This happens often. Here's how to handle it:

| Situation | Approach |
|-----------|----------|
| Official source vs. community source | Go with official, but note if community consistently reports differently |
| Old official source vs. recent community | Note uncertainty, suggest verifying directly |
| Different community sources disagree | Note that experiences vary |
| Can't verify at all | Don't include, or clearly mark as unverified |

### Example:
> "According to CDCR policy, visitors may bring up to $70 in coins and bills. Some visitors report that certain facilities are stricter about bill denominations — check with your facility if unsure."

---

## Information That Changes Frequently

Be especially careful with:

| Topic | Why It Changes | How to Handle |
|-------|----------------|---------------|
| Phone/video providers | Contracts change | Note "as of [date]" |
| Visiting schedules | Facility discretion | Link to CDCR for current |
| Package vendors | Approved list updates | Direct to official list |
| Pricing | Vendors adjust | Give ranges, not exact |
| Facility status | Lockdowns, closures | Always say "verify before visiting" |

## Citing Sources in Content

### For General Guides
- List sources at the end of each guide
- Include URLs where helpful
- Note "Information current as of [Month Year]"

### For Specific Claims
If stating something specific (a price, a rule, a deadline):
- Ensure you can point to where you found it
- If from community source, frame appropriately

### You Don't Need to Cite
- General knowledge (prisons have visiting hours)
- Common procedures described on CDCR site
- Your own synthesis of multiple sources

---

## Keeping Information Current

### Monthly Tasks
- Check CDCR news page for policy changes
- Scan Prison Talk for reported changes
- Verify vendor websites for pricing/service changes

### When Users Report Errors
- Verify the correction
- Update content promptly
- Note the update date

### Major Changes to Watch For
- New phone/video vendor contracts (happens every few years)
- Legislative changes affecting visits/communication
- Facility openings/closings/reclassifications
