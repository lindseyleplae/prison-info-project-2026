# Prison Visitor Guide — Project Instructions

## What This Is
A national reference website (prisonvisitorguide.org) for families of incarcerated people. Factual, organized information about prison visiting procedures, communication, costs, and rights — state by state, facility by facility.

## The #1 Rule: Tone
**Strictly informational. Like Wikipedia.** No emotional language, no advice-giving, no therapy speak, no condescension, no overuse of "loved one." This is non-negotiable. See `scripts/lint-tone.mjs` for the enforced banned phrase list.

**Read [SCALING.md](SCALING.md) at the start of any session that creates or modifies more than one content file.** It is the project's institutional memory — every rule in it exists because someone broke it and Lindsey caught it. Skipping it means repeating prior mistakes.

## Critical Rules
1. All content tone must be strictly informational (run `npm run lint:tone`)
2. Every fact is sourced or attributed — never invented. Every guide and facility page cites `sources:` (enforced by `check:sources`). When in doubt, leave it out.
3. All styling goes through CSS custom properties — never hardcoded color values
4. Adding a state or facility = content files plus the two data entries in PLAYBOOK Step 5 (`state-visiting-rules.ts`, `system-source-labels.ts`), nothing else
5. Verify-before-relying language is layout-injected on every facility page; state guides end with a "Verify Before Acting" callout
6. No live status claims — reference info only, never real-time data
7. Mobile-first — majority of users are on phones, often with bad signal
8. WCAG 2.1 AA minimum accessibility (enforced by `npm run check:a11y`)
9. Judgment calls already made live in DECISIONS.md — follow them, don't re-decide them

- Iowa (Iowa DOC): 6 state guides + 9 facility pages (complete) — Iowa's Department of Corrections runs nine adult prisons, ALL state-operated (no private prisons, no out-of-state placement). Term "incarcerated individual" (the formally adopted term; the website still mixes in "offender"). The nine: Iowa State Penitentiary [ISP, Fort Madison — the maximum-security men's prison; oldest (1839 origins), rebuilt 2015], Anamosa State Penitentiary [ASP — medium men; redesignated from max in 2023 when max moved to ISP; the Luster Heights minimum prison farm is a satellite], Iowa Medical and Classification Center [IMCC, Coralville/Johnson County (the "Oakdale" campus) — the men's reception/classification center + the system's medical/psychiatric hub + the Iowa Forensic Psychiatric Hospital], Mount Pleasant Correctional Facility [MPCF — minimum men's reentry since 2016 (men-only; the historical women's unit closed then — do NOT say it holds women); hosts the statewide Centralized Visiting Authority], Newton Correctional Facility [medium + the statewide minimum-security Correctional Release Center], North Central Correctional Facility [Rockwell City — minimum men], Clarinda Correctional Facility [medium men; Special Needs/Intensive Supervision mission — NOT geriatric; not the closed "Clarinda Academy" juvenile site], Fort Dodge Correctional Facility [medium men; RIVERS reentry program], and the Iowa Correctional Institution for Women [ICIW, Mitchellville — the ONLY women's prison + the women's reception center; full min/med/max range, 888-bed campus]. NO death penalty (abolished 1965) — no death-row content anywhere. Reception: men at IMCC (Coralville), women at ICIW (Mitchellville). Three distinctive vendor facts: incoming personal mail goes OFF-SITE to Pigeonly (Las Vegas, NV; P.O. Box 96777, 89193) for scanning since 2022, addressed with the person's ID + a per-facility numeric code (only ISP=1110 and Anamosa=1106 confirmed; legal mail to the facility; photos via Pelipost); communications run through Ameelio, a NONPROFIT provider used across all nine prisons (video, voice, and messaging, offered free to families — the legacy ICSolutions per-minute data conflicts, so no rate is asserted); and medical care is STATE-RUN/in-house — Iowa considered privatizing in 2025 and rejected it (Nov 2025), so there is no for-profit medical contractor. Money via Access Corrections / JPay / Western Union. Visiting: an approved list (cap of six plus immediate family) via the Centralized Visiting Authority at Mount Pleasant (mailed application), then scheduled per facility through the Ameelio Connect app (≥4 days/week). Independent oversight is the Iowa Office of Ombudsman (with an assistant ombudsman for corrections). The eight Judicial District Departments of Correctional Services (community-based corrections — probation/parole/residential) are a separate tier, noted not paged.
